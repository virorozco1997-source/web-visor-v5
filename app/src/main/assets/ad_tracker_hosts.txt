# AdBlocker - lista básica de dominios de publicidad y rastreo (trackers).
# Un dominio por línea. Las líneas que empiezan con # son comentarios.
# Los subdominios se bloquean automáticamente (ej. pagead2.googlesyndication.com).
#
# Esto NO es un adblocker completo estilo uBlock Origin / EasyList (esos
# manejan decenas de miles de reglas, incluyendo ocultamiento de elementos
# por CSS). Esta lista cubre las redes de publicidad y analítica más
# comunes a nivel de dominio: basta para eliminar la mayoría de los
# banners, videos publicitarios y trackers de comportamiento en sitios
# comunes, pero algún sitio muy dependiente de su propio ad-server puede
# igual mostrar algo.
